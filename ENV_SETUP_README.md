# N-AutoPost 환경 설정

## API 키 설정 방법

1. 프로젝트 루트 디렉토리에 `.env.local` 파일을 생성하세요
2. 아래 내용을 복사하여 붙여넣으세요:

```
VITE_GEMINI_API_KEY=여기에_실제_API_키를_입력하세요
```

3. https://ai.google.dev/ 에 접속하여 Gemini API 키를 발급받으세요
4. 발급받은 API 키를 위 파일에 입력하세요
5. 서버를 재시작하세요 (`Ctrl+C` 후 `pnpm run dev`)

## 문제 해결

### "인기 뉴스 불러오기"가 작동하지 않는 경우

1. `.env.local` 파일이 생성되었는지 확인
2. API 키가 올바르게 입력되었는지 확인
3. 서버를 재시작했는지 확인
4. 브라우저 콘솔(F12)에서 에러 메시지 확인

### 포트 충돌 문제

현재 포트 3000에 다른 애플리케이션이 실행 중입니다.
- 다른 서버를 종료하거나
- `vite.config.ts`에서 포트를 변경하세요 (예: 3001, 5173 등)
